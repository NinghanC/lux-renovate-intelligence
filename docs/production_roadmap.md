# Production Roadmap

## Month 1

- Replace file storage with PostgreSQL/PostGIS and object storage.
- Add a managed vector index and retrieval evaluation set.
- Add OCR for scanned PDFs and drawings.
- Add prompt, evidence, and generation audit logs.

## Month 2

- Ingest SECO historical inspection reports, defect observations, photos, measurements, and project metadata.
- Build a unified defect and inspection taxonomy.
- Add document-level permissions and role-based access control.
- Add human feedback capture on dossier sections.

## Month 3

- Similar historical case retrieval.
- Recurring defect pattern analytics.
- Experience-enhanced mission checklist.
- Human-reviewed report draft assistance.
- Monitoring for retrieval quality, validation failures, and model drift.
