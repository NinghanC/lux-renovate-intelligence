# Upload Test Documents

These files are synthetic and are intended only for testing the local upload flow.

Use them from the frontend upload button:

1. Select a demo site.
2. Upload one of these files.
3. Click `Generate`.
4. Check whether uploaded-document evidence appears in the Evidence Panel.

Recommended files:

- `old_building_owner_notes.md`: tests Markdown parsing and missing-document detection.
- `preliminary_site_visit_notes.txt`: tests plain-text parsing and checklist generation.
- `sample_condition_observations.pdf`: tests PDF upload parsing.
